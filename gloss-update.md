# Glossary update suggestions

High-priority (appears in experience/projects and may confuse non-technical recruiters):

- MVP (Minimum Viable Product): a first, pared-down version of a product used to validate ideas quickly with real users.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- Monorepo: a single repository that contains multiple related projects or services.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- RAG (Retrieval-Augmented Generation): combines a search step with an LLM so answers can cite or use relevant documents.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- FastAPI: a Python web framework for building fast APIs and backend services.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- Protocol Buffers: a compact data format used to serialize structured data, often with gRPC.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- Docker Hub: a registry for storing and sharing Docker container images.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- PyTorch: a popular machine learning framework for training deep-learning models.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- spaCy: an NLP library for building language processing pipelines.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json
- GitHub: a platform for hosting code repositories and collaborating on software.
  Used in: src/content/experience/data.json, src/content/experience/data.de.json, src/content/projects/data.json
